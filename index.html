Quick Start 
// Connect to DocumentDB
const db = new DocumentDB({
  endpoint: "your-cluster-endpoint",
  credentials: "your-credentials"
});

// Create a collection
const collection = await db.createCollection("users");

// Insert a document
const user = await collection.insertOne({
  name: "John Doe",
  email: "john@example.com",
  age: 30
});

console.log("Hello World from DocumentDB!");

Install DocumentDB

Get started with our simple installation process.

2

Connect to Database

Establish a connection to your DocumentDB instance.

const db = new DocumentDB({ endpoint: "localhost:27017" });

Start Building

Create collections, insert documents, and build your application.

const result = await collection.insertOne({ message: "Hello World!" });
